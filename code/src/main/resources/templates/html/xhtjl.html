合同记录使用说明
基本操作
  • 点击对应列表单元格可以直接修改信息，修改后数据自动保存
  • 点击新增按钮可以录入新信息
  • 单击对应行，变成浅蓝色为选中状态，点击删除可执行删除操作
  • 在有选中行的情况下，点击下方生成退货单按钮可以获取对应信息生成退货单
注意事项
  • 在新增内容时对账状态一般情况下不需要操作，采取默认值就可以

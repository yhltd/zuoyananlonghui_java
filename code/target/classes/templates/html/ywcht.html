已完成合同使用说明
基本操作
  • 页面加载自动更新已完成的合同内容
  • 可根据业务单位名称进行条件查询
  • 点击刷新可以重新查询所有已完成合同。
  • 点击工序配置可以在弹窗内调整工序单价配置，调整完需要点击对应行的保存按钮。
  • 在表格内单击可以选中该行，之后点击删除可以删除选择的合同记录。
  • 表格内单元格可以单击选择，进行修改，修改后自动保存。
注意事项
  • 删除后数据无法恢复
